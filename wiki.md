##  项目描述

### 1 数据迁移 tags-etl 
    编写代码将mysql数据表数据迁移到HBase表中
   
### 2 标签模型调度 tags-oozie
     编写代码，Oozie Java Client API 如何调度执行workFlow / Coordinator
     
### 3 工具类 tags-up
    实现模型调度，主要HDFS文件系统操作Oozie Java Client API 使用工具类封装

### 4 tags-web 
     用户画像标签系统的web平台
     
### 5 tags-model
     模型开发，给用户打标签，每个业务标签就开发SparkApplication程序，调度执行
